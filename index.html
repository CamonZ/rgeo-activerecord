RGeo::ActiveRecord

RGeo ActiveRecord extensions and tools for spatial connection adapters

Download this project as a .zip file Download this project as a tar.gz file

RGeo is a key component for writing location-aware applications in the Ruby programming language. At its core is an implementation of the industry standard OGC Simple Features Specification, which provides data representations of geometric objects such as points, lines, and polygons, along with a set of geometric analysis operation.

RGeo::ActiveRecord is an optional RGeo module providing spatial extensions for ActiveRecord, as well as a set of helpers for writing spatial ActiveRecord adapters based on RGeo.

Adapters that use RGeo include:

  • activerecord-postgis-adapter for PostGIS
  • activerecord-spatialite-adapter for Spatialite
  • activerecord-mysql2spatial-adapter for MySQL Spatial
